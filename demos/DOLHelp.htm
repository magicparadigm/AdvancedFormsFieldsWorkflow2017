How is DocuSign Responding?

DocuSign has been working with a group of financial services customers to discover their thoughts on the new DOL rule and what changes need to be made to the product to support their needs. Based on this feedback, one key feature that is being added to DocuSign is the ability to add supplemental materials (such as disclosures and other informational documents) that need to accompany a document sent for signature. These supplemental documents are available to the signer to view and acknowledge, without making the envelope too large or confusing for signers.  Currently this functionality is only accessible via the API and is not usable from the interactive web application (NDSE). The product is in BETA right now and not available in the production environment. This feature is still undergoing testing and revision.

 

Creating a DocuSign Envelope with Supplemental Documents

      At this time, it is not possible to interactively create a DocuSign envelope with the new supplemental documents functionality. The DOL web site was created to allow DocuSign employees to create envelopes that utilize the supplemental documents functionality. Here are the steps to create a DocuSign envelope containing one main document and one supplemental document:

 

1.  To launch the DOL web site, go to the URL: http://sedemo1.cloudapp.net/advancedformfields/demos/DOL.aspx

 

2.  Enter the signer information manually or click on the text: Dept of Labor Fiduciary Rule (DOL) to populate some default values.

 

You have the option of changing the DocuSign account used to create the envelope by filling in the account information section.  By default, the DocuSign account used for the TGK Financial Services portal will be used.

 

3.  Choose the document to be signed in the envelope by clicking on the Choose File button. Once a file has been selected, click on the Upload button to upload it to the server.  

 

 

4.  Choose the location a signature tab by specifying the tab name, page, xPosition and yPosition


 

5.  Choose the supplemental document to be used in the envelope by clicking the Choose File button and once selected, use the Upload button to upload the file to the server.


 

6.  Select the options used for the supplemental document.  

 

 

 

 

 

 

 

7.  The signing session will be displayed with the document to sign in the main window


 

 

8.  The supplemental document will be specified either in a modal window or separate window based on whether you selected Modal or New Browser Window as the display option when you click on the View button. If you selected Inline as the display option, it will look like a regular document in the main window
